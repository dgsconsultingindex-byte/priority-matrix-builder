Logo Usage Guidelines

Follow these rules to maintain brand consistency and quality.

✓ Do

  • Use official logo files provided
  • Maintain clear space around the logo
  • Use on contrasting backgrounds
  • Maintain aspect ratio when scaling
  • Use high-resolution versions for print
  • Follow minimum size requirements

✗ Don't

  • Distort, stretch, or skew the logo
  • Change logo colors or add effects
  • Place on busy backgrounds
  • Rotate or tilt the logo
  • Recreate or modify the logo
  • Use low-resolution versions

Minimum Sizes

Never reproduce the logo smaller than these dimensions to ensure legibility.

Digital Applications

  • Primary Logo: 120px width minimum
  • Wordmark: 100px width minimum
  • Icon: 32px × 32px minimum
  • Favicon: 16px × 16px (icon only)

Print Applications

  • Primary Logo: 1 inch (2.54cm) width
  • Stacked Logo: 0.75 inch (1.9cm) width
  • Business Cards: 0.5 inch (1.27cm) width
  • Letterhead: 1.5 inch (3.81cm) width
